igrep.c is a command line tool built to imitate grep without regular expression capability. Behavior can be modified with the use of the following
flags/options:

-i: Ignore case 
-n: Print line numbers 
-h: Help

Compile:
```
$ gcc -o igrep igrep.c
```

Execute:
```
$ ./igrep [OPTION]... FILE
```
